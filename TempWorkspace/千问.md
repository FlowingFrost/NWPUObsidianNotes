### 第四篇 热学

#### 第一章 气体动理论

##### §1 分子热运动与统计规律

|概念|关键公式 / 描述|备注|
|:--|:--|:--|
|**热学研究对象**|热现象（宏观） ↔ 热运动（微观）|热运动是大量微观粒子的无规则运动|
|**宏观量 ↔ 微观量**|宏观量 = 微观量统计平均|例：压强 $p$、温度 $T$ 为宏观量；分子速度 $v$、动能 $E_k$ 为微观量|
|**三大基本观点**|① 物质由大量分子组成，分子间有空隙  <br>② 分子永不停息做无规则热运动（扩散、布朗运动）  <br>③ 分子间存在引力和斥力，平衡距离 $d_0\sim10^{-10}\ \mathrm{m}$|布朗运动剧烈程度 $\propto T$|
|**统计规律性**|大量偶然事件 → 必然分布（伽尔顿板）|概率归一：$\displaystyle\sum_i P_i=1$ 或 $\displaystyle\int f(x)dx=1$|
|**统计平均值**|离散：$\langle x\rangle=\sum_i x_i P_i$  <br>连续：$\langle x\rangle=\int x f(x)dx$|$f(x)$ 为概率密度|

##### §2 理想气体状态方程

|概念|关键公式 / 描述|备注|
|:--|:--|:--|
|**理想气体模型**|① 分子线度可忽略（质点）  <br>② 除碰撞外无相互作用  <br>③ 弹性碰撞|适用条件：低压、高温|
|**状态参量**|压强 $p$，体积 $V$，温度 $T$|平衡态：宏观量不随时间变化|
|**理想气体状态方程**|$$pV = \nu RT = \frac{m}{M_{\text{mol}}}RT$$|$R=8.31\ \mathrm{J/(mol\cdot K)}$  <br>$\nu$：物质的量|
|**分子数形式**|$$p = nkT$$|$n=\frac{N}{V}$：分子数密度  <br>$k=\frac{R}{N_A}=1.38\times10^{-23}\ \mathrm{J/K}$|
|**摩尔质量关系**|$$M_{\text{mol}} = N_A m_0$$|$m_0$：单个分子质量|

##### §3 理想气体的压强和温度的统计意义

|概念|关键公式 / 描述|备注|
|:--|:--|:--|
|**压强统计意义**|推导链：  <br>单个分子冲量 → 多个分子冲量 → 冲量统计值 → 压力 → 压强|$$p=\frac{1}{3}nm_{0}\overline{v^2}=\frac{2}{3}n\overline{E_k}$$|
|**温度微观意义**|联立 $p=nkT$ 和 $p=\frac{2}{3}n\overline{E_k}$ 得：  <br>$$\overline{E_k} = \frac{3}{2}kT$$|温度标志着物体内部分子无规则运动的剧烈程度，是其平均平动动能的量度|
|**方均根速率**|$$v_{rms} = \sqrt{\overline{v^2}} = \sqrt{\frac{3kT}{m_{0}}} = \sqrt{\frac{3RT}{M_{\text{mol}}}}$$|适用于计算分子的平均能量|

##### §4 能量按自由度均分定理

|概念|关键公式 / 描述|备注|
|:--|:--|:--|
|**自由度 (i)**|确定物体空间位置所需独立坐标数|刚性分子不考虑振动|
|**刚性分子自由度**|单原子：$i=3$ (平动)  <br>双原子：$i=5$ (3平动+2转动)  <br>多原子：$i=6$ (3平动+3转动)||
|**能量均分定理**|在温度为 $T$ 的平衡态下，分子每个自由度的平均动能均为 $\frac{1}{2}kT$||
|**分子平均动能**|$$\overline{E_k} = \frac{i}{2}kT$$||
|**理想气体内能**|$$E = \frac{i}{2} \nu RT = \frac{i}{2} \frac{m}{M_{\text{mol}}} RT$$|结论：理想气体内能只与温度有关，是温度的单值函数|

##### §5 气体分子数按速率分布的统计规律

|概念|关键公式 / 描述|备注|
|:--|:--|:--|
|**速率分布函数 $f(v)$**|$$f(v) = \frac{1}{N} \frac{dN}{dv}$$  <br>归一化：$$\int_0^\infty f(v) dv = 1$$|表示速率在 $v$ 附近单位速率区间内的分子数占总分子数的比率|
|**麦克斯韦速率分布**|$$f(v) = 4\pi \left( \frac{m_0}{2\pi kT} \right)^{3/2} v^2 e^{-\frac{m_0v^2}{2kT}}$$|平衡态下理想气体的速率分布|
|**三种特征速率**|最概然速率：$v_p = \sqrt{\frac{2kT}{m_0}}$  <br>平均速率：$\bar{v} = \sqrt{\frac{8kT}{\pi m_0}}$  <br>方均根速率：$v_{rms} = \sqrt{\frac{3kT}{m_0}}$|$v_p < \bar{v} < v_{rms}$  <br>最概然速率适用于讨论速率分布|

##### §6 玻尔兹曼能量分布律

|概念|关键公式 / 描述|备注|
|:--|:--|:--|
|**玻尔兹曼分布律**|粒子数 $\propto e^{-E/kT}$|在温度为 $T$ 的平衡态下，系统处于能量 $E$ 状态的概率与 $e^{-E/kT}$ 成正比|
|**重力场中按高度分布**|$$n = n_0 e^{-\frac{m_0 g z}{kT}}$$  <br>等温气压公式：$$p = p_0 e^{-\frac{M_{\text{mol}} g z}{RT}}$$|随高度升高，气体越稀薄，压强越低|

##### §7 平均自由程和平均碰撞频率

| 概念                          | 关键公式 / 描述                                                                                                                | 备注                                                                   |
| :-------------------------- | :----------------------------------------------------------------------------------------------------------------------- | :------------------------------------------------------------------- |
| **平均自由程 ($\bar{\lambda}$)** | 一个分子连续两次碰撞间路程的平均值                                                                                                        |                                                                      |
| **平均碰撞频率 ($\bar{Z}$)**      | 一个分子单位时间内与其他分子碰撞次数的平均值                                                                                                   |                                                                      |
| **表达式**                     | $$\bar{Z} = \sqrt{2} \pi d^2 \bar{v} n$$  <br>$$\bar{\lambda} = \frac{\bar{v}}{\bar{Z}} = \frac{1}{\sqrt{2} \pi d^2 n}$$ | $d$ 为分子有效直径  <br>结论：$\bar{\lambda}$ 与 $n$ (或 $p$) 成反比，与 $\bar{v}$ 无关 |
