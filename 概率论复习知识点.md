## 第二章
### 分布函数
称F(x) = P{X ≤ x}, -∞ < x < +∞为随机变量X的分布函数。记作X ～ F(x)或X ～ F_X(x)。
(1) 有界性：对任意实数x，有0 ≤ F(x) ≤ 1
(2) 单调性：F(x)是单调不减函数，即对任意x₁ < x₂，有F(x₁) ≤ F(x₂)
(3) 极限性质：F(-∞) = lim[x→-∞] F(x) = 0，F(+∞) = lim[x→+∞] F(x) = 1
(4) 右连续性：F(x)是右连续的，即对任意实数a，lim[x→a⁺] F(x) = F(a)
设F(x)是随机变量X的分布函数，则：
- P{X ≤ b} = F(b) 
- P{a < X ≤ b} = F(b) - F(a)
- P{X > a} = 1 - F(a)
- P{X < b} = F(b-0)（其中F(b-0)表示F(x)在点b处的左极限）
- P{X = b} = F(b) - F(b-0)

分布函数的计算:
1. 离散型随机变量 : 分布律 $P\{X=x_i\}=P_i$    $\sum\limits_{i=1}^{k}P\{X=x_i\}=1$
2. 连续型随机变量 : 密度函数$p(x)$    $\int_{-\infty}^{+\infty}p(x)dx = 1$
### 密度函数
*累计*分布函数是对密度函数的积分

### 一维随机变量及其分布
#### 离散型随机变量分布
分布函数  F(x) = P{X ≤ x} = ∑[xₖ ≤ x] P{X = xₖ}
##### 二项分布 $X \sim B(n,p)$
$P\{X = k\} = C_{n}^{k}p^{k}(1-p)^{(n-k)}, k = 0,1,2,...,n$   其中0 ≤ p ≤ 1，则称X服从参数为(n,p)的二项分布，记作X ~ B(n,p)。
##### 泊松分布 $X\sim\pi(\lambda)$  
基于二项分布，n足够大($n\geq20$)且p较小($p\leq0.05$)
$P\{X = k\} =\large{\frac{\lambda^{k}}{k!}} e^{-\lambda},(\lambda=np)k = 0,1,2,...,$  ==$\lambda=np$ 可以视为期望==

**泊松定理**：设Xₙ ~ B(n, pₙ)，且$\lim\limits_{n→∞} npₙ = λ > 0$，则对任意非负整数k，有$\lim\limits_{n→∞} P\{Xₙ = k\} = \lim\limits_{n→∞} C_n^kp_n^k(1-pₙ)^{(n-k)} = \frac{λ^{k}}{k!}e^{(-λ)}$
这个定理表明泊松分布可作为二项分布的极限分布。当n很大，p很小，而λ = np适中时，可用泊松分布近似计算二项分布。
##### 超几何分布 $X\sim H(N,n,M)$
**不放回，N个产品含次品M个，不放回地抽n个抽出次品的个数记为X**
$P\{X=k\}=\large{\frac{C^k_{M}C^{n-k}_{N-M}}{C^n_N}}$  $(k=0,1,2,...,min\{M,n\})$
#### 连续型随机变量
##### 指数分布 $X\sim Exp(\lambda)$
**事件A在单位时间内发生次数的期望为 $\lambda$ 则其发生所用时间X服从指数分布**
可以表示寿命，具有无记忆性($P\{X > s + t \mid X > s\} = P\{X > t\},\quad \forall s,t \geq 0$)
密度函数$p(x)=\begin{cases}\lambda e^{-\lambda x},&x\geq 0\\0,&x<0 \end{cases}$

##### 均匀分布$X\sim U[a,b]$
在区间上等可能取值的随机变量
密度函数$p(x)=\begin{cases} \frac{1}{{b-a}},&a\leq x\leq b \\0,&\text{其他} \end{cases}$

##### 正态分布 $X\sim N(\mu,\sigma^2)$
密度函数 $\phi(x) = \frac{1}{\sqrt{2\pi}\sigma} e^{-\frac{(x-\mu)^2}{2\sigma^2}}, \quad -\infty < x < +\infty$
$\mu$ 是曲线对称轴   $\sigma$ 越小曲线分布越集中    标准正态分布$X\sim N(0,1)$
**标准正态分布变换**: $X\sim N(\mu,\sigma^2)$  $Y\sim N(0,1)$ $Y=\frac{X-\mu}{\sigma}$

## 第三章 随机变量的数字特征

### 数学期望

**1. 定义**：
- 离散型随机变量：$E(X) = \sum x_k \cdot p_k$，其中$P\{X=x_k\}=p_k$
- 连续型随机变量：$E(X) = \int_{-\infty}^{+\infty} x \cdot p(x) dx$，其中$p(x)$是密度函数
- 数学期望反映随机变量所有取值的加权平均，权重为对应概率

**2. 常见分布的数学期望**：
- 0-1分布 $X \sim B(1,p)$：$E(X) = p$
- 二项分布 $X \sim B(n,p)$：$E(X) = np$
- 泊松分布 $X \sim P(\lambda)$：$E(X) = \lambda$
- 均匀分布 $X \sim U[a,b]$：$E(X) = \frac{a+b}{2}$
- 正态分布 $X \sim N(\mu,\sigma^2)$：$E(X) = \mu$
- 指数分布 $X \sim Exp(\lambda)$：$E(X) = \frac{1}{\lambda}$

**3. 随机变量函数的数学期望**：
- 一元函数：$E[g(X)] = \sum g(x_k) \cdot p_k$（离散）或 $E[g(X)]=\int g(x) \cdot p(x)dx$（连续）
- 二元函数：$E[g(X,Y)] = \sum \sum g(x_i,y_j) \cdot p_{ij}$（离散）或 $E[g(X,Y)]=\int \int g(x,y) \cdot p(x,y)dxdy$（连续）

**4. 数学期望的性质**：
- $E(C) = C$（常数性质）
- $E(CX) = CE(X)$（常数因子）
- $E(X+Y) = E(X) + E(Y)$（线性性质）
- $E(XY) = E(X)E(Y)$（当X,Y相互独立时）

### 方差

**1. 定义**：
- 方差：$D(X) = E[X-E(X)]^2$，反映随机变量取值偏离期望的程度
- 计算公式：$D(X) = E(X^2) - [E(X)]^2$
- 标准差：$\sigma(X) = \sqrt{D(X)}$

**2. 常见分布的方差**：
- 0-1分布 $X \sim B(1,p)$：$D(X) = p(1-p)$
- 二项分布 $X \sim B(n,p)$：$D(X) = np(1-p)$
- 泊松分布 $X \sim P(\lambda)$：$D(X) = \lambda$
- 均匀分布 $X \sim U[a,b]$：$D(X) = \frac{(b-a)^2}{12}$
- 正态分布 $X \sim N(\mu,\sigma^2)$：$D(X) = \sigma^2$
- 指数分布 $X \sim Exp(\lambda)$：$D(X) = \frac{1}{\lambda^2}$

**3. 方差的性质**：
- $D(C) = 0$（常数性质）
- $D(CX) = C^2D(X)$（常数因子）
- $D(X+Y) = D(X) + D(Y)$（当X,Y相互独立时）

**4. 切比雪夫不等式**：
- $P\{|X-\mu| \geq \varepsilon\} \leq \frac{\sigma^2}{\varepsilon^2}$，其中$\mu=E(X), \sigma^2=D(X)$

### 协方差与相关系数

**1. 协方差**：
- 定义：$cov(X,Y) = E[(X-E(X))(Y-E(Y))]$
- 计算公式：$cov(X,Y) = E(XY) - E(X)E(Y)$
- 性质：当X,Y相互独立时，$cov(X,Y)=0$

**2. 相关系数**：
- 定义：$\rho_{XY} = \frac{cov(X,Y)}{\sqrt{D(X)} \cdot \sqrt{D(Y)}}$
- 性质：$|\rho_{XY}| \leq 1$
- 线性相关：$\rho_{XY} = \pm 1$表示X,Y完全线性相关
- 不相关的充要条件：$\rho_{XY} = 0$，即$E(XY)=E(X)E(Y)$
- 独立与不相关的区别：X,Y相互独立 $\Rightarrow$ X,Y不相关（$\rho_{XY} = 0$），但反之不成立

## 第四章 大数定律与中心极限定理

### 随机变量序列的收敛性

**1. 依概率收敛**：
- 定义：随机变量序列{Yn}依概率收敛于随机变量Y，记作$Y_n \xrightarrow{P} Y$，若对任意$\varepsilon > 0$，$\lim_{n \to \infty}P\{|Y_n - Y| \geq \varepsilon\} = 0$

**2. 依分布收敛**：
- 定义：随机变量序列{Yn}的分布函数为Fn(x)，随机变量Y的分布函数为F(x)，若对F(x)的所有连续点x，都有$\lim_{n \to \infty}F_n(x) = F(x)$，则称{Yn}依分布收敛于Y，记作$Y_n \xrightarrow{L} Y$

**3. 收敛性强弱关系**：几乎处处收敛/ r阶收敛 $\Rightarrow$ 依概率收敛 $\Rightarrow$ 依分布收敛

### 大数定律

**1. 大数定律的定义**：
- 若{Xn}为随机变量序列，{an}为常数序列
- 设$Y_n = \frac{1}{n}\sum_{i=1}^{n}X_i$
- 对任意$\varepsilon > 0$，当$n \to \infty$时，$P\{|Y_n - a_n| < \varepsilon\} \to 1$，或$Y_n \xrightarrow{P} a_n$
- 则称{Xn}服从大数定律

**2. 切比雪夫大数定律**：
- 条件：随机变量序列{Xn}两两不相关，方差存在且有共同上界，即$D(X_i) \leq C$ (i = 1,2,...)
- 结论：算术平均值依概率收敛于期望的平均值，即$\frac{1}{n}\sum_{i=1}^{n}X_i \xrightarrow{P} \frac{1}{n}\sum_{i=1}^{n}E(X_i)$

**3. 辛钦大数定律**：
- 条件：随机变量序列{Xn}独立同分布，期望存在，即$E(X_i) = \mu$ (i = 1,2,...)
- 结论：算术平均值依概率收敛于共同期望，即$\frac{1}{n}\sum_{i=1}^{n}X_i \xrightarrow{P} \mu$

**4. 伯努利大数定律**：
- 条件：n重伯努利试验，事件A发生的次数为$\mu_n$，事件A在每次试验中发生的概率为p
- 结论：频率$\mu_n/n$依概率收敛于概率p，即$\frac{\mu_n}{n} \xrightarrow{P} p$

### 中心极限定理

**1. 林德贝格-列维中心极限定理（独立同分布中心极限定理）**：
- 条件：随机变量X1, X2, ..., Xn相互独立同分布，$E(X_i) = \mu, D(X_i) = \sigma^2 \neq 0$
- 结论：标准化变量$\frac{\sum_{i=1}^{n}X_i - n\mu}{\sigma\sqrt{n}}$渐近服从标准正态分布，即$\frac{\sum_{i=1}^{n}X_i - n\mu}{\sigma\sqrt{n}} \xrightarrow{L} N(0,1)$
- 推论1：当n充分大时，$\sum_{i=1}^{n}X_i \sim N(n\mu, n\sigma^2)$
- 推论2：当n充分大时，$\frac{1}{n}\sum_{i=1}^{n}X_i \sim N(\mu, \frac{\sigma^2}{n})$

**2. 棣莫弗-拉普拉斯中心极限定理**：
- 条件：随机变量Yn ~ B(n,p)（二项分布）
- 结论：标准化变量$\frac{Y_n - np}{\sqrt{np(1-p)}}$的分布函数的极限为标准正态分布函数
- 意义：当n很大时，二项分布可用正态分布近似，即$B(n,p) \sim N(np, np(1-p))$
