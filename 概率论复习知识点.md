## 第二章
### 分布函数
称F(x) = P{X ≤ x}, -∞ < x < +∞为随机变量X的分布函数。记作X ～ F(x)或X ～ F_X(x)。
(1) 有界性：对任意实数x，有0 ≤ F(x) ≤ 1
(2) 单调性：F(x)是单调不减函数，即对任意x₁ < x₂，有F(x₁) ≤ F(x₂)
(3) 极限性质：F(-∞) = lim[x→-∞] F(x) = 0，F(+∞) = lim[x→+∞] F(x) = 1
(4) 右连续性：F(x)是右连续的，即对任意实数a，lim[x→a⁺] F(x) = F(a)
设F(x)是随机变量X的分布函数，则：
- P{X ≤ b} = F(b) 
- P{a < X ≤ b} = F(b) - F(a)
- P{X > a} = 1 - F(a)
- P{X < b} = F(b-0)（其中F(b-0)表示F(x)在点b处的左极限）
- P{X = b} = F(b) - F(b-0)

分布函数的计算:
1. 离散型随机变量 : 分布律 $P\{X=x_i\}=P_i$    $\sum\limits_{i=1}^{k}P\{X=x_i\}=1$
2. 连续型随机变量 : 密度函数$p(x)$    $\int_{-\infty}^{+\infty}p(x)dx = 1$
### 密度函数
*累计*分布函数是对密度函数的积分

### 一维随机变量及其分布
#### 离散型随机变量分布
分布函数  F(x) = P{X ≤ x} = ∑[xₖ ≤ x] P{X = xₖ}
##### 二项分布 $X \sim B(n,p)$
$P\{X = k\} = C_{n}^{k}p^{k}(1-p)^{(n-k)}, k = 0,1,2,...,n$   其中0 ≤ p ≤ 1，则称X服从参数为(n,p)的二项分布，记作X ~ B(n,p)。
##### 泊松分布 $X\sim\pi(\lambda)$  
基于二项分布，n足够大($n\geq20$)且p较小($p\leq0.05$)
$P\{X = k\} =\large{\frac{\lambda^{k}}{k!}} e^{-\lambda},(\lambda=np)k = 0,1,2,...,$  ==$\lambda=np$ 可以视为期望==

**泊松定理**：设Xₙ ~ B(n, pₙ)，且$\lim\limits_{n→∞} npₙ = λ > 0$，则对任意非负整数k，有$\lim\limits_{n→∞} P\{Xₙ = k\} = \lim\limits_{n→∞} C_n^kp_n^k(1-pₙ)^{(n-k)} = \frac{λ^{k}}{k!}e^{(-λ)}$
这个定理表明泊松分布可作为二项分布的极限分布。当n很大，p很小，而λ = np适中时，可用泊松分布近似计算二项分布。
##### 超几何分布 $X\sim H(N,n,M)$
**不放回，N个产品含次品M个，不放回地抽n个抽出次品的个数记为X**
$P\{X=k\}=\large{\frac{C^k_{M}C^{n-k}_{N-M}}{C^n_N}}$  $(k=0,1,2,...,min\{M,n\})$
#### 连续型随机变量
##### 指数分布 $X\sim Exp(\lambda)$
**事件A在单位时间内发生次数的期望为 $\lambda$ 则其发生所用时间X服从指数分布**
可以表示寿命，具有无记忆性($P\{X > s + t \mid X > s\} = P\{X > t\},\quad \forall s,t \geq 0$)
密度函数$p(x)=\begin{cases}\lambda e^{-\lambda x},&x\geq 0\\0,&x<0 \end{cases}$

##### 均匀分布$X\sim U[a,b]$
在区间上等可能取值的随机变量
密度函数$p(x)=\begin{cases} \frac{1}{{b-a}},&a\leq x\leq b \\0,&\text{其他} \end{cases}$

##### 正态分布 $X\sim N(\mu,\sigma^2)$
密度函数 $\phi(x) = \frac{1}{\sqrt{2\pi}\sigma} e^{-\frac{(x-\mu)^2}{2\sigma^2}}, \quad -\infty < x < +\infty$
$\mu$ 是曲线对称轴   $\sigma$ 越小曲线分布越集中    标准正态分布$X\sim N(0,1)$
**标准正态分布变换**: $X\sim N(\mu,\sigma^2)$  $Y\sim N(0,1)$ $Y=\frac{X-\mu}{\sigma}$

## 第三章 随机变量的数字特征

### 数学期望
- 离散型随机变量：$E(X) = \sum x_k \cdot p_k$，其中$P\{X=x_k\}=p_k$
- 连续型随机变量：$E(X) = \int_{-\infty}^{+\infty} x \cdot p(x) dx$，其中$p(x)$是密度函数

**常见分布的数学期望**：
- 0-1分布 $X \sim B(1,p)$：$E(X) = p$
- 二项分布 $X \sim B(n,p)$：$E(X) = np$
- 泊松分布 $X \sim P(\lambda)$：$E(X) = \lambda$
- 均匀分布 $X \sim U[a,b]$：$E(X) = \frac{a+b}{2}$
- 正态分布 $X \sim N(\mu,\sigma^2)$：$E(X) = \mu$
- 指数分布 $X \sim Exp(\lambda)$：$E(X) = \frac{1}{\lambda}$

**3. 随机变量函数的数学期望**：==(来源:随机变量函数的概率分布和密度)==
- 一元函数：$E[g(X)] = \sum g(x_k) \cdot p_k$（离散）或 $E[g(X)]=\int g(x) \cdot p(x)dx$（连续）
- 二元函数：$E[g(X,Y)] = \sum \sum g(x_i,y_j) \cdot p_{ij}$（离散）或 $E[g(X,Y)]=\int \int g(x,y) \cdot p(x,y)dxdy$（连续）

**4. 数学期望的性质**：
- $E(C) = C$（常数性质）
- $E(CX) = CE(X)$（常数因子）
- $E(X+Y) = E(X) + E(Y)$（线性性质）
- $E(XY) = E(X)E(Y)$（当X,Y相互独立时）

### 方差
- 方差：$D(X) = E[X-E(X)]^2$，反映随机变量取值偏离期望的程度
- 计算公式：$D(X) = E(X^2) - [E(X)]^2$
- 标准差：$\sigma(X) = \sqrt{D(X)}$

**2. 常见分布的方差**：
- 0-1分布 $X \sim B(1,p)$：$D(X) = p(1-p)$
- 二项分布 $X \sim B(n,p)$：$D(X) = np(1-p)$
- 泊松分布 $X \sim P(\lambda)$：$D(X) = \lambda$
- 均匀分布 $X \sim U[a,b]$：$D(X) = \frac{(b-a)^2}{12}$
- 正态分布 $X \sim N(\mu,\sigma^2)$：$D(X) = \sigma^2$
- 指数分布 $X \sim Exp(\lambda)$：$D(X) = \frac{1}{\lambda^2}$

**3. 方差的性质**：
- $D(C) = 0$（常数性质）
- $D(CX) = C^2D(X)$（常数因子）
- $D(X+Y) = D(X) + D(Y)$（当X,Y相互独立时）

**4. 切比雪夫不等式**：
- $P\{|X-\mu| \geq \varepsilon\} \leq \frac{\sigma^2}{\varepsilon^2}$，其中$\mu=E(X), \sigma^2=D(X)$

### 协方差与相关系数

**1. 协方差**：
- 定义：$cov(X,Y) = E[(X-E(X))(Y-E(Y))]$
- 计算公式：$cov(X,Y) = E(XY) - E(X)E(Y)$
- 性质：当X,Y相互独立时，$cov(X,Y)=0$

**2. 相关系数**：
- 定义：$\rho_{XY} = \frac{cov(X,Y)}{\sqrt{D(X)} \cdot \sqrt{D(Y)}}=\frac{cov(X,Y)}{\rho_x\cdot\rho_y}$
- 性质：$|\rho_{XY}| \leq 1$
- 线性相关：$\rho_{XY} = \pm 1$表示X,Y完全线性相关
- 不相关的充要条件：$\rho_{XY} = 0$，即$E(XY)=E(X)E(Y)$
- 独立与不相关的区别：X,Y相互独立 $\Rightarrow$ X,Y不相关（$\rho_{XY} = 0$），但反之不成立

## 第四章 大数定律与中心极限定理
随机变量序列 $X_{n}\sim B(n,0.5)$   $Y_{n}=\frac{1}{n}\sum\limits_{i=0}^{n}X_i$  取值和概率是两回事


### 随机变量序列的收敛性

**1. 依概率收敛**：
- 定义：随机变量序列{Yn}依概率收敛于随机变量Y，记作$Y_n \xrightarrow{P} Y$，若对任意$\varepsilon > 0$，$\lim_{n \to \infty}P\{|Y_n - Y| \geq \varepsilon\} = 0$

**2. 依分布收敛**：
- 定义：随机变量序列{Yn}的分布函数为Fn(x)，随机变量Y的分布函数为F(x)，若对F(x)的所有连续点x，都有$\lim_{n \to \infty}F_n(x) = F(x)$，则称{Yn}依分布收敛于Y，记作$Y_n \xrightarrow{L} Y$

**3. 收敛性强弱关系**：几乎处处收敛/ r阶收敛 $\Rightarrow$ 依概率收敛 $\Rightarrow$ 依分布收敛

### 大数定律

**1. 大数定律的定义**：
- 若{Xn}为随机变量序列，{an}为常数序列
- 设$Y_n = \frac{1}{n}\sum_{i=1}^{n}X_i$
- 对任意$\varepsilon > 0$，当$n \to \infty$时，$P\{|Y_n - a_n| < \varepsilon\} \to 1$，或$Y_n \xrightarrow{P} a_n$
- 则称{Xn}服从大数定律

**2. 切比雪夫大数定律**：
- 条件：随机变量序列{Xn}两两不相关，方差存在且有共同上界，即$D(X_i) \leq C$ (i = 1,2,...)
- 结论：算术平均值依概率收敛于期望的平均值，即$\frac{1}{n}\sum_{i=1}^{n}X_i \xrightarrow{P} \frac{1}{n}\sum_{i=1}^{n}E(X_i)$

**3. 辛钦大数定律**：
- 条件：随机变量序列{Xn}独立同分布，期望存在，即$E(X_i) = \mu$ (i = 1,2,...)
- 结论：算术平均值依概率收敛于共同期望，即$\frac{1}{n}\sum_{i=1}^{n}X_i \xrightarrow{P} \mu$

**4. 伯努利大数定律**：
- 条件：n重伯努利试验，事件A发生的次数为$\mu_n$，事件A在每次试验中发生的概率为p
- 结论：频率$\mu_n/n$依概率收敛于概率p，即$\frac{\mu_n}{n} \xrightarrow{P} p$

### 中心极限定理

**1. 林德贝格-列维中心极限定理（独立同分布中心极限定理）**：
- 条件：随机变量X1, X2, ..., Xn相互独立同分布，$E(X_i) = \mu, D(X_i) = \sigma^2 \neq 0$
- 结论：标准化变量$\frac{\sum_{i=1}^{n}X_i - n\mu}{\sigma\sqrt{n}}$渐近服从标准正态分布，即$\frac{\sum_{i=1}^{n}X_i - n\mu}{\sigma\sqrt{n}} \xrightarrow{L} N(0,1)$
- 推论1：当n充分大时，$\sum_{i=1}^{n}X_i \sim N(n\mu, n\sigma^2)$
- 推论2：当n充分大时，$\frac{1}{n}\sum_{i=1}^{n}X_i \sim N(\mu, \frac{\sigma^2}{n})$

**2. 棣莫弗-拉普拉斯中心极限定理**：
- 条件：随机变量Yn ~ B(n,p)（二项分布）
- 结论：标准化变量$\frac{Y_n - np}{\sqrt{np(1-p)}}$的分布函数的极限为标准正态分布函数
- 意义：当n很大时，二项分布可用正态分布近似，即$B(n,p) \sim N(np, np(1-p))$


## 第五章 数理统计的基本概念

### 第一节 总体、样本与统计量

#### 一、总体与样本
- **总体**：研究对象的全体，可以看作一个随机变量X
- **个体**：组成总体的每一个单元
- **样本**：从总体X中抽取的n个个体X₁, X₂, ..., Xₙ，满足：
  1. X₁, X₂, ..., Xₙ相互独立
  2. X₁, X₂, ..., Xₙ都与总体X同分布
- **样本容量**：n，表示样本中个体的数量
- **样本观测值**：样本取得的一组具体数值(x₁, x₂, ..., xₙ)

#### 二、统计量
- **定义**：设X₁, X₂, ..., Xₙ是来自总体X的一个样本，g(X₁, X₂, ..., Xₙ)是样本的函数，若g中不含任何未知参数，则称g(X₁, X₂, ..., Xₙ)为一个统计量
- **常用统计量**：
  - **样本均值**：$\overline{X} = \frac{1}{n}\sum_{i=1}^{n}X_i$
  - **样本方差**：$S^2 = \frac{1}{n-1}\sum_{i=1}^{n}(X_i - \overline{X})^2$
  - **样本k阶原点矩**：$A_k = \frac{1}{n}\sum_{i=1}^{n}X_i^k$
  - **样本k阶中心矩**：$B_k = \frac{1}{n}\sum_{i=1}^{n}(X_i - \overline{X})^k$
  - **样本标准差**：$S = \sqrt{S^2}$

#### 三、次序统计量
- **定义**：将样本观测值按大小顺序排列得到新的样本
- **最大次序统计量**：$X_{(n)} = \max(X_1, X_2, ..., X_n)$
- **最小次序统计量**：$X_{(1)} = \min(X_1, X_2, ..., X_n)$
- **第k个次序统计量**：$X_{(k)}$，表示第k小的值

#### 四、经验分布函数
- **定义**：根据样本观测值构造的分布函数，用于估计总体分布函数
- **格里文科定理**：当样本容量趋于无穷大时，经验分布函数将几乎处处收敛于总体的分布函数

### 第二节 抽样分布（常用统计分布）

#### 一、χ²分布（卡方分布）
- **定义**：设X₁, X₂, ..., Xₙ相互独立，都服从标准正态分布N(0,1)，则称随机变量$\chi^2 = X_1^2 + X_2^2 + ... + X_n^2$服从自由度为n的χ²分布，记作χ² ~ χ²(n)
- **概率密度函数**：当x > 0时，$f(x) = \frac{1}{2^{n/2}\Gamma(n/2)}x^{n/2-1}e^{-x/2}$
- **性质**：
  - 数学期望：E(χ²) = n
  - 方差：D(χ²) = 2n
  - 可加性：若χ₁² ~ χ²(n₁)，χ₂² ~ χ²(n₂)，且χ₁²与χ₂²独立，则χ₁² + χ₂² ~ χ²(n₁ + n₂)

#### 二、t分布（学生氏分布）
- **定义**：设X ~ N(0,1)，Y ~ χ²(n)，且X与Y相互独立，则称随机变量$T = \frac{X}{\sqrt{Y/n}}$服从自由度为n的t分布，记作T ~ t(n)
- **性质**：
  - 当n充分大时，t分布近似于标准正态分布
  - t分布的图形关于t = 0对称
  - E(T) = 0 (n > 1)
  - D(T) = n/(n-2) (n > 2)

#### 三、F分布
- **定义**：设U ~ χ²(n₁)，V ~ χ²(n₂)，且U与V相互独立，则称随机变量$F = \frac{U/n_1}{V/n_2}$服从自由度为(n₁, n₂)的F分布，记作F ~ F(n₁, n₂)
- **性质**：
  - 若F ~ F(n₁, n₂)，则1/F ~ F(n₂, n₁)
  - 若T ~ t(n)，则T² ~ F(1, n)

#### 四、正态总体的样本均值和样本方差的分布
- **定理1**：设X₁, X₂, ..., Xₙ是来自正态总体N(μ, σ²)的样本，则有：
  1. 样本均值$\overline{X} \sim N(\mu, \sigma^2/n)$
  2. 样本均值$\overline{X}$与样本方差$S^2$相互独立
  3. $\frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$
- **定理2**（t分布的t统计量）：在定理1的条件下，$\frac{\overline{X}-\mu}{S/\sqrt{n}} \sim t(n-1)$
- **两个正态总体的抽样分布**：
  - $\frac{\overline{X}-\overline{Y}-(\mu_1-\mu_2)}{\sqrt{\frac{\sigma_1^2}{n_1}+\frac{\sigma_2^2}{n_2}}} \sim N(0,1)$
  - $\frac{S_1^2/\sigma_1^2}{S_2^2/\sigma_2^2} \sim F(n_1-1, n_2-1)$
  - 当σ₁² = σ₂² = σ²时，$\frac{(\overline{X}-\overline{Y})-(\mu_1-\mu_2)}{S_w\sqrt{\frac{1}{n_1}+\frac{1}{n_2}}} \sim t(n_1+n_2-2)$
    其中$S_w^2 = \frac{(n_1-1)S_1^2+(n_2-1)S_2^2}{n_1+n_2-2}$称为合并方差

## 第六章 参数估计

### 第一节 参数的点估计

#### 一、矩估计法
- **基本思想**：用样本的k阶原点矩$A_k = \frac{1}{n}\sum_{i=1}^{n}X_i^k$去估计总体的k阶原点矩$E(X^k)$
- **求解步骤**：
  1. 求出$E(X^k) = \alpha_k(\theta_1, \theta_2, ..., \theta_m)$，$k=1,2,...,m$
  2. 令$A_k = \alpha_k$，$k=1,2,...,m$
  3. 解出$\hat{\theta_1}, \hat{\theta_2}, ..., \hat{\theta_m}$
- **特点**：方法简单，但有时估计效果不佳

#### 二、最大似然估计法
- **基本思想**：使样本出现的可能性最大，即在已知样本观测值的情况下，选择使该样本出现概率最大的参数值
- **似然函数**：设总体X的分布律为$P(X=x) = p(x;\theta)$或分布密度为$p(x;\theta)$，似然函数为$L(\theta) = \prod_{i=1}^{n}p(x_i;\theta)$
- **求解步骤**：
  1. 求似然函数$L(\theta)$
  2. 求出$\ln L(\theta)$及似然方程$\frac{\partial \ln L(\theta)}{\partial \theta_i} = 0$，$i=1,2,...,m$
  3. 解似然方程得到最大似然估计值
  4. 将估计值中的样本观测值换成随机变量，得到最大似然估计量
- **特点**：在大样本下具有优良的统计性质

#### 三、估计量的优良性准则
- **无偏性**：设$\hat{\theta} = \hat{\theta}(X_1, X_2, ..., X_n)$是参数$\theta$的一个估计量，如果$E(\hat{\theta}) = \theta$，则称$\hat{\theta}$是$\theta$的无偏估计量
- **渐近无偏性**：如果$\lim_{n \to \infty}E(\hat{\theta_n}) = \theta$，则称$\hat{\theta_n}$是$\theta$的渐近无偏估计量
- **有效性**：若对任意$\theta$的无偏估计量$\hat{\theta_1}$和$\hat{\theta_2}$，均有$D(\hat{\theta_1}) < D(\hat{\theta_2})$，则称$\hat{\theta_1}$比$\hat{\theta_2}$有效
- **相合性（一致估计）**：设$\hat{\theta_n}$是$\theta$的一个估计量，如果依概率收敛于$\theta$，即对任意$\varepsilon > 0$，有$\lim_{n \to \infty}P\{|\hat{\theta_n} - \theta| \geq \varepsilon\} = 0$，则称$\hat{\theta_n}$是$\theta$的相合估计

### 第二节 参数的区间估计

#### 一、基本概念
- **置信区间**：设总体$X \sim F(x;\theta)$，$\theta$为未知参数，$(X_1, X_2, ..., X_n)$是来自总体X的样本。如果存在两个统计量$\hat{\theta_1}(X_1, X_2, ..., X_n)$和$\hat{\theta_2}(X_1, X_2, ..., X_n)$（$\hat{\theta_1} < \hat{\theta_2}$），使得对于给定的$\alpha (0 < \alpha < 1)$，有$P\{\hat{\theta_1} \leq \theta \leq \hat{\theta_2}\} = 1-\alpha$，则称区间$[\hat{\theta_1}, \hat{\theta_2}]$为$\theta$的置信度为$1-\alpha$的置信区间

#### 二、正态总体均值的置信区间
- **方差$\sigma^2$已知，求$\mu$的置信区间**：
  - 统计量：$U = \frac{\overline{X}-\mu}{\sigma/\sqrt{n}} \sim N(0,1)$
  - 置信区间：$[\overline{X} - \frac{\sigma}{\sqrt{n}}u_{\alpha/2}, \overline{X} + \frac{\sigma}{\sqrt{n}}u_{\alpha/2}]$
- **方差$\sigma^2$未知，求$\mu$的置信区间**：
  - 统计量：$T = \frac{\overline{X}-\mu}{S/\sqrt{n}} \sim t(n-1)$
  - 置信区间：$[\overline{X} - \frac{S}{\sqrt{n}}t_{\alpha/2}(n-1), \overline{X} + \frac{S}{\sqrt{n}}t_{\alpha/2}(n-1)]$

#### 三、正态总体方差的置信区间
- **求$\sigma^2$的置信区间**：
  - 统计量：$\chi^2 = \frac{(n-1)S^{*2}}{\sigma^2} \sim \chi^2(n-1)$
  - 置信区间：$[\frac{(n-1)S^{*2}}{\chi^2_{\alpha/2}(n-1)}, \frac{(n-1)S^{*2}}{\chi^2_{1-\alpha/2}(n-1)}]$

#### 四、两个正态总体参数的区间估计
- **均值差的置信区间**：
  - 方差已知：$[(\overline{X}-\overline{Y}) \pm u_{\alpha/2}\sqrt{\frac{\sigma_1^2}{n}+\frac{\sigma_2^2}{m}}]$
  - 方差未知但相等：$[(\overline{X}-\overline{Y}) \pm t_{\alpha/2}(n+m-2)S_w\sqrt{\frac{1}{n}+\frac{1}{m}}]$
- **方差比的置信区间**：
  - $\sigma_1^2/\sigma_2^2$的置信区间：$[\frac{S_1^{*2}}{S_2^{*2}} \cdot \frac{1}{F_{\alpha/2}(n-1,m-1)}, \frac{S_1^{*2}}{S_2^{*2}} \cdot \frac{1}{F_{1-\alpha/2}(n-1,m-1)}]$

## 第七章 假设检验

### 第一节 假设检验的基本概念

#### 一、基本概念
- **原假设（零假设）H₀**：待检验的假设
- **备择假设 H₁**：与原假设对立的假设
- **显著性水平 α**：α = P{拒绝H₀ | H₀正确}，即犯第一类错误的概率
- **检验统计量**：用于检验假设的统计量
- **拒绝域**：拒绝原假设H₀的所有样本值所组成的集合
- **两类错误**：
  - 第一类错误（弃真错误）：当H₀为真时拒绝H₀，概率为α
  - 第二类错误（取伪错误）：当H₀为假时接受H₀，概率为β

#### 二、检验的基本思想
- **基本原理**：小概率事件原理，即小概率事件在一次试验中几乎不可能发生
- **基本步骤**：
  1. 提出待检验的假设H₀及备择假设H₁
  2. 选择适当的统计量，在H₀成立的条件下确定它的概率分布
  3. 给定检验水平α，确定临界值和拒绝域
  4. 根据样本观察值计算统计量的值
  5. 判断统计量值是否落入拒绝域，作出拒绝或接受H₀的判断

### 第二节 正态总体参数的假设检验

#### 一、单个正态总体参数的检验

**1. 已知方差时均值的检验（U检验）**
- 假设：H₀: μ = μ₀，H₁: μ ≠ μ₀
- 统计量：$U = \frac{\overline{X}-\mu_0}{\sigma/\sqrt{n}} \sim N(0,1)$（H₀为真时）
- 拒绝域：$W = \{|u| \geq u_{\alpha/2}\}$

**2. 未知方差时均值的检验（t检验）**
- 假设：H₀: μ = μ₀，H₁: μ ≠ μ₀
- 统计量：$T = \frac{\overline{X}-\mu_0}{S/\sqrt{n}} \sim t(n-1)$（H₀为真时）
- 拒绝域：$W = \{|t| \geq t_{\alpha/2}(n-1)\}$

**3. 未知均值时方差的检验（χ²检验）**
- 假设：H₀: σ² = σ₀²，H₁: σ² ≠ σ₀²
- 统计量：$\chi^2 = \frac{(n-1)S^2}{\sigma_0^2} \sim \chi^2(n-1)$（H₀为真时）
- 拒绝域：$W = \{\chi^2 \leq \chi^2_{1-\alpha/2}(n-1) \text{ 或 } \chi^2 \geq \chi^2_{\alpha/2}(n-1)\}$

#### 二、两个正态总体参数的检验

**1. 已知方差时两总体均值差的检验（U检验）**
- 假设：H₀: μ₁ = μ₂，H₁: μ₁ ≠ μ₂
- 统计量：$U = \frac{\overline{X}-\overline{Y}}{\sqrt{\sigma_1^2/n_1+\sigma_2^2/n_2}} \sim N(0,1)$（H₀为真时）
- 拒绝域：$W = \{|u| \geq u_{\alpha/2}\}$

**2. 未知但相等方差时两总体均值差的检验（t检验）**
- 假设：H₀: μ₁ = μ₂，H₁: μ₁ ≠ μ₂
- 统计量：$T = \frac{\overline{X}-\overline{Y}}{S_w\sqrt{1/n_1+1/n_2}} \sim t(n_1+n_2-2)$（H₀为真时）
  其中$S_w^2 = \frac{(n_1-1)S_1^2+(n_2-1)S_2^2}{n_1+n_2-2}$
- 拒绝域：$W = \{|t| \geq t_{\alpha/2}(n_1+n_2-2)\}$

**3. 两总体方差比的检验（F检验）**
- 假设：H₀: σ₁² = σ₂²，H₁: σ₁² ≠ σ₂²
- 统计量：$F = \frac{S_1^2}{S_2^2} \sim F(n_1-1,n_2-1)$（H₀为真时）
- 拒绝域：$W = \{f \leq F_{1-\alpha/2}(n_1-1,n_2-1) \text{ 或 } f \geq F_{\alpha/2}(n_1-1,n_2-1)\}$

#### 三、单侧检验
- **左侧检验**：H₀: μ ≥ μ₀ vs H₁: μ < μ₀，拒绝域为$\{统计量 \leq 临界值\}$
- **右侧检验**：H₀: μ ≤ μ₀ vs H₁: μ > μ₀，拒绝域为$\{统计量 \geq 临界值\}$
- **双侧检验**：H₀: μ = μ₀ vs H₁: μ ≠ μ₀，拒绝域为$\{|统计量| \geq 临界值\}$
