==先分析已知内容，列公式==
1. 建立自然坐标系与加速度分析 摆锤的切向加速度$a_t$和法向加速度$a_n$： 
	$a_t = l\ddot{\varphi}, \quad a_n = l\dot{\varphi}^2$
2. 列质点动力学微分方程
	切向（$t$方向）:  $m a_t = \frac{W}{g} l\ddot{\varphi} = -W\sin\varphi \tag{1}$ 
	法向（$n$方向):   $m a_n = \frac{W}{g} l\dot{\varphi}^2 = F_N - W\cos\varphi \tag{2}$ 
	其中$F_N$为摆线的拉力。 
==代换$\phi$，先统一变量然后代换==
3. 对切向方程做变量代换与积分 利用变量代换$\ddot{\varphi} = \dot{\varphi}\frac{d\dot{\varphi}}{d\varphi} = \frac{1}{2}\frac{d\dot{\varphi}^2}{d\varphi}$，代入式(1)得： $$ \frac{1}{2}\frac{d\dot{\varphi}^2}{d\varphi} = -\frac{g}{l}\sin\varphi $$ 对其积分==（初始条件：$\varphi = \varphi_0$时，$\dot{\varphi} = 0$；末态$\varphi$时，$\dot{\varphi}$为角速度）==： $$ \int_{0}^{\dot{\varphi}^2} d(\dot{\varphi}^2) = \int_{\varphi_0}^{\varphi} -\frac{2g}{l}\sin\varphi \, d\varphi $$ 积分结果： $$ \dot{\varphi}^2 = \frac{2g}{l}(\cos\varphi - \cos\varphi_0) \tag{4} $$ 
4. 求摆线拉力并确定最大值 将式(4)代入式(2)，化简得拉力表达式： $$ F_N = W(3\cos\varphi - 2\cos\varphi_0) $$ 当摆锤到达**最低位置**（$\varphi = 0$）时，$\cos\varphi$取最大值1，此时拉力$F_N$最大： $$ F_{N\text{max}} = W(3 - 2\cos\varphi_0) $$ 
核心结论 摆线中的最大拉力为$\boldsymbol{F_{N\text{max}} = W(3 - 2\cos\varphi_0)}$，出现在摆锤运动到竖直最低位置时。

### 11.1 Work done by a force
1. 常力的功
2. work done by a variable force
	元功 $\Delta{W}=F\cos{\alpha}ds=F_{\tau}ds=\bar{F}\cdot d\bar{r}=F_Xdx+F_Ydy+F_Zdx$
	功：积分 $W=\int_{\overrightarrow{M_1}}^{\overrightarrow{M_2}}\vec{F}\cdot d\vec{r}$   $\vec{M_1}$ 是初始位置(位矢，数值上和坐标相同)，$\vec{M_2}$是末位置
3. Work done by the resultant force
	合力的功等于各个分力功的代数和
	$W=\int\limits_{M_1}^{M_2}\vec{F_合}\cdot d\vec{r}$$
4. Work done by special forces
	1. by gravity：$W=Mgh$
	2. by an elastic force弹力：$W=\frac{k}{2}(\delta_1^2-\delta_2^2)$ 为什么用小写δ？
	3. by the gravitational force of celestial bodies : $W=GMm(\frac{1}{r_1}-\frac{1}{r_2})$	
		**Conservative force**保守力：做功只与始末位置有关。
	4. ==Work done by forces applied to a rotating body==
		![[Pasted image 20251230161714.png]]
		$F_\tau$ 对转动有帮助，做功。另外两个方向分力不做功
		注：$m_z(\bar{F})$是以z为轴的力矩
		力对于物体的矩乘以$d\phi$
		![[Pasted image 20251230162031.png|200]]
	5. Work done by friction force
		力的元功等于力乘以作用点的元位移。
		所以纯滚动时，摩擦力不做功
5. Work done by internal force
	内力之和为0，做功之和
	AB两点：$\delta W=\bar{F}\cdot d(BA)$
	rigibody：内力做功之和为0
	一般物体：不为0
	**最常见的内力做功**：内燃机
6. Work done by reaction forces of constraints:
	理想状态下，约束力不做功。

## D4 principle of angular momentum

| 平动  | 转动   |
| --- | ---- |
| 速度  | 角速度  |
| 加速度 | 角加速度 |
| 质量  | 转动惯量 |
| 力   | 力矩   |
| 动量  | 角动量  |

$M_o(\overrightarrow{F})=\overrightarrow{r}\times\overrightarrow{F}$    力矩=力臂 叉乘 力
角动量 $mv$ 关于O点可以定义为A对O的矩 $M_o(m\overrightarrow{v})=\overrightarrow{r}\times{m}\overrightarrow{v}$    `动量矩=位置矢量 叉乘 动量`
$M_X(m\overrightarrow{v})=y(mv_z)-z(mv_y)$ 原因：$v_{y}$ $v_z$垂直于x轴，且x轴垂直纸面向外时，$z*v_y$ 向右为顺时针，$y*v_z$ 向上为逆时针

#### Angular momentum of particle system
**about point**：$L_o=\sum\limits{M_o(m_iv_i)}$
**about axis**： $L_X=\sum\limits{M_X(m_iv_i)}$

#### Angular momentum of ranslation rigid body about fixed point O
$L_o=r_C\times{Mv_C}$

#### Angular momentum of rotation rigid body about its axis
$M_z(mv)=r_z\cdot{m}\,r_z\omega=m{r_z}^2\omega$
$L_z=\sum\limits{M_z(m_iv_i)}=\sum\limits{m_i{r_{zi}}^2\omega}=\omega\sum\limits{m_i{r_{zi}}^2}=J_z\omega$ , $J$ 转动惯量

#### Angular momentum of a rigid body in plane motion
$L_o=r_c\times{Mv_c}+L_c$ 平动+质心定轴转动

### D4.2 Principle of angular momentum
1. Angular momentum theorem for a particle
	$\frac{dL_o}{dt}=\sum\limits{M_o(F_i^{(e)})}$ 动量矩定理
2. about a fixed axis
	$\frac{dL_X}{dt}=\sum\limits{M_X(F_i^{(e)})}$
3. for a particle system with respect to tis mass center
	$\frac{dL_{C_r}}{dt}=\sum\limits{M_C(F_i^{(e)})}=M_c^{(e)}$
- Conservation law of angular momentum
	1. about a fixed point O
		若外力矩为0，则$L_o=const$
	2. about a fixed axis
		若外力矩为0，则$L_X=const$
	3. about the mass center C
		若外力矩为0，则$L_{C_r}=const$
