## Chapter6. Kinematics of a Particle
### 6.1 Description of the Motion of a Particle Using the Method of Position Vector  
使用位置矢量法描述质点的运动

- **运动的分解 / composition of motion**：复杂运动可视为多个简单运动的合成  
- **轨迹 / trajectory**：质点在空间中运动所经过的路径  
- **运动的相对性 / relativity of motion**：运动描述依赖于所选参考系  
  - **参考物体 / reference object**  
  - **参考系 / reference system**  
- **时间描述**  
  - **时刻 / instant of time**  
  - **时间间隔 / interval of time**  
- **力学模型 / mechanics model**  
  - **质点 / particles**：忽略大小和形状，只考虑质量的理想化模型  
    - Linear motion（直线运动）  
    - Curvilinear motion（曲线运动）
  - 刚体/Rigid bodies
    - Basic motion
      - translation
      - rotation
    - planar motion  

#### 矢量法（Vector Method）
- **位矢 / position vector**：$\vec{r} = \vec{r}(t)$  
- **位移 / displacement**：$\Delta \vec{r}$  
- **运动方程 / motional equation**：$\vec{r} = \vec{r}(t)$

#### 坐标法（Rectangular Coordinates）
- $\vec{r} = x(t)\,\mathbf{i} + y(t)\,\mathbf{j} + z(t)\,\mathbf{k}$  
- 速度：$\vec{v} = \frac{d\vec{r}}{dt} = \dot{x}\,\mathbf{i} + \dot{y}\,\mathbf{j} + \dot{z}\,\mathbf{k}$  
- 加速度：$\vec{a} = \frac{d^2\vec{r}}{dt^2} = \ddot{x}\,\mathbf{i} + \ddot{y}\,\mathbf{j} + \ddot{z}\,\mathbf{k}$  

> **注**：常用直角坐标法和自然坐标法；矢量法多用于理论推导。

#### 自然坐标法（Natural Coordinates）
- 弧坐标 $s = s(t)$：沿轨迹从某起点量取的路径长度  
- **曲率**：$k = \lim_{\Delta s \to 0} \frac{\Delta \theta}{\Delta s}$  
- **曲率半径**：$\rho = \frac{1}{k}$  
- 自然坐标系由切向 $\tau$、法向 $n$、副法向 $b$ 构成
