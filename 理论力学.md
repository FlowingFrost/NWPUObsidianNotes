### 11.1 Work done by a force
1. 常力的功
2. work done by a variable force
	元功 $\Delta{W}=F\cos{\alpha}ds=F_{\tau}ds=\bar{F}\cdot d\bar{r}=F_Xdx+F_Ydy+F_Zdx$
	功：积分
3. Work done by the resultant force
	合力的功等于各个分力功的代数和
	$W=\int\limits_{M_1}^{M_2}$
4. Work done by special forces
	1. by gravity：$W=Mgh$
	2. by an elastic force弹力：$W=\frac{k}{2}(\delta_1^2-\delta_2^2)$ 为什么用小写δ？
	3. by the gravitational force
	**Conservative force**保守力：做功只与始末位置有关。
	4. Work done by forces applied to a rotating body
		$F_\tau$ 对转动有帮助，做功。另外两个方向分力不做功
		公式：...
		注：$m_z(\bar{F})$是力矩
		力对于物体的矩乘以$d\phi$
	5. Work done by friction force
		力的元功等于力乘以作用点的元位移。
		所以纯滚动时，摩擦力不做功
5. Work done by internal force
	内力只和为0，做功之和
	AB两点：$\delta W=\bar{F}\cdot d(BA)$
	rigibody：内力做功之和为0
	一般物体：不为0
	**最常见的内力做功**：内燃机
6. Work done by reaction forces of constraints:
	理想状态下，支持力不做功。
