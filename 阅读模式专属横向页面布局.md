## 新增功能说明：

### 1. 手动分栏标记

在 Markdown 中使用：

markdown

<!-- 基本分栏标记 -->
```<div class="column-break"></div>```

<!-- 在元素前分栏 -->
```<div class="column-break-before">接下来内容在新栏</div>```

<!-- 在元素后分栏 -->
```<div class="column-break-after">这个元素后分栏</div>```

### 2. 页面大小控制

**在设置中调整：**

- 页面宽度缩放：0-200%
    
- 页面高度缩放：0-200%
    

**使用预设类名：**

css

/* 小尺寸 */
.page-size-small

/* 大尺寸 */  
.page-size-large

/* 全宽模式 */
.page-size-fullwidth

/* 紧凑模式 */
.page-size-compact

### 3. 分栏区域控制

markdown

<!-- 确保内容块不被分割 -->
<div class="column-region">
这部分内容会保持在同一栏内
</div>

<!-- 跨所有栏 -->
<div class="column-span-all">
这个内容会跨越所有栏
</div>

### 4. 响应式设计

在移动设备（宽度 ≤ 768px）上自动切换为单栏布局，确保移动端阅读体验。

这些新增功能让你可以更精细地控制分栏布局，适应不同的内容和阅读需求。
